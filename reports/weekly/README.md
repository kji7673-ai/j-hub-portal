# 주간 업무 계획서 저장소

## 저장 규칙
1. **폴더:** `reports/weekly/YYYY-WNN/` (ISO 주차 기준)
2. **파일명:** `본부명_주간계획.md`
3. **작성 기한:** 매주 금요일 오후 5시
4. **템플릿:** `templates/주간계획서_템플릿.md` 참조

## 자동 연동
- `run_daily_sync.sh` 실행 시 이 폴더의 파일이 자동으로 파싱되어 J-Hub 포털에 반영됩니다.
- 향후 네이버 밴드 API 연동 시 `BAND_ACCESS_TOKEN` 환경변수만 설정하면 자동 수집됩니다.

## 폴더 구조
```
reports/weekly/
├── README.md
├── templates/
│   └── 주간계획서_템플릿.md
├── 2026-W29/

│   ├── 도시개발본부_주간계획.md
│   └── ...
└── 2026-W30/
    └── ...
```
